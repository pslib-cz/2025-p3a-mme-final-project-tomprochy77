## Projekt: Rotační Stromový Kalendář
# Popis
Mechanický věčný kalendář kombinující laserem řezanou překližku a 3D tištěné funkční jádro. Mechanismus využívá 3D tištěná vnitřní vodítka (offset 0,3 mm), která nahrazují složité lepení dřevěných vrstev a zajišťují hladkou rotaci datových prstenců. Podstavec je rovněž 3D tištěný pro maximální stabilitu a přesné usazení panelu.
